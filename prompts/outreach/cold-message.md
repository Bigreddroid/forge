# Cold Outreach Message Generator

## System Prompt

```
You write cold outreach messages for B2B founders. Your messages get replies because they are:
- Specific (reference something real about the person or company)
- Short (under 75 words)
- Value-first (no pitch in the first message)
- Human (not AI-sounding, not template-sounding)

Never:
- Start with "I hope this message finds you well"
- Lead with what you do or sell
- Ask for a call in the first message
- Use "synergy", "leverage", "touch base", or "circle back"
- Sound like you copy-pasted this to 500 people (even if you did)
```

## User Prompt Template

```
Write a cold outreach message for LinkedIn / Email using:

Prospect info:
- Name: [first name]
- Title: [their role]
- Company: [company name]
- Observation: [something specific — a post they wrote, a recent hire, a product launch, a company milestone]

My context:
- I am: [your name, founder of X]
- My angle: [the one insight or resource that's relevant to them]
- Goal of this message: [start a conversation / share a resource / no ask]

Message type: [LinkedIn connection note / LinkedIn DM / cold email]
```

## Message Templates

### LinkedIn Connection Note (300 char max)

```
Saw your post on [specific topic] — [one-sentence genuine reaction].
Building something similar on the [growth/content/systems] side.
Would love to connect.
```

### LinkedIn DM (First Message — Value Only)

```
Hey [Name],

[Specific observation about their work in 1 sentence].

I've been thinking about [related insight] — wrote something short on it last week that might be relevant: [link or 1-sentence summary].

No ask — just thought it'd be useful given what you're working on.

[Your name]
```

### Cold Email (First Touch)

```
Subject: [specific thing relevant to them — not "quick question"]

Hey [Name],

[1-sentence observation about a specific thing they're doing or building].

[1-sentence relevant insight, stat, or story that connects to their situation].

Not pitching anything — just thought this was relevant given [specific reason].

[Your name]
[Title, Company]
```

### Cold Email (Follow-up if No Reply)

```
Subject: Re: [original subject]

Hey [Name],

Wanted to follow up on this — particularly given [timely or specific reason it's relevant now].

[One-sentence value add or update].

Worth a quick chat?

[Your name]
```
